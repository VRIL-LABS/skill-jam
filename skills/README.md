## Organization Method

Skills were automatically categorized based on their naming patterns and content focus:
- Document type detection (PDF, DOCX, etc.)
- Platform/framework identification (iOS, React, Angular, etc.)
- Functional area (security, search, AI, etc.)
- Technology stack (databases, browsers, blockchain, etc.)

Each skill is extracted into its own subfolder within the appropriate category, preserving the original archive structure.

## Usage

Navigate to any category folder to explore skills in that domain:

```bash
cd skills-staging/web-frontend/
cd skills-staging/ios-mobile/
cd skills-staging/search-research/
```

All original zip files remain in the repository root for reference.
